# objectos :: process 

A simple Java 8 Process manager.

## Maven

objectos :: process is at Maven central.

```xml
<dependency>
    <groupId>br.com.objectos</groupId>
    <artifactId>process</artifactId>
    <version>0.1.0</version>
</dependency>
```
